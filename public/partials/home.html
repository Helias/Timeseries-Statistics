Timeseries Statistics

Questo tool permette di analizzare dati statistici, in particolare serie storiche

Filtrando i dati, analizzando il trend, rimuovendo eventuali rumori ed effettuando una previsione per ottenere un prospetto dei dati futuri

Questa applicazione web si appoggia alle Google API per mostrare i dati tramite line chart.
